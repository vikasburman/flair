Overview

OOJS (pronounced ojus) is a standalone JavaScript library that brings powerful object oriented and associated concepts, that are available in languages like C# and Java, into JavaScript as native JavaScript implementation. No transpilation is required for these OOPS concepts to work; its all pure JavaScript. These powerful features are:

  • Classes and Interfaces
  • Simple and Multiple Inheritance
  • Structures and Enumerations
  • Assemblies
  • Aspect Oriented Programming
  • Attributes Based Programming
  • Dependency Injection
  • Serialization
  • Reflection

To use this library, just two steps are needed:

  1. Include oojs.min.js
  2. oojs({...options...})
